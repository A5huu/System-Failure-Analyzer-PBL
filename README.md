# System-Failure-Analyzer-PBL
The 2nd sem C PBL project for the degree of BCA(hons.) with Artificial Intelligence and Data Science at Graphic Era Hill University, Bhimtal.
System Failure Analyzer is a C based system monitoring and diagnosis tool.

## Features
- System Parameter Analysis
- Risk Score Calculation
- Automatic log generation
- Timestamped diagnostic reports

## Future Improvement
- Log history analyzer
- Failure/maintenance prediction
- Alert system

## Project Structure
System-Failure-Analyzer-PBL
│
├── src
│   ├── pbl.c
│   └── sys.c
│
├── README.md
└── .gitignore


## File Structure 
- **sys.c**
   Used to develop, test, and debug individual functions or modules before integration.
- **pbl.c**
   Main program that integrates the tested and working features from `sys.c` to build the complete tool.

## Development Workflow

1. Implement and test new functionality in `sys.c`.
2. Verify the feature works correctly.
3. Integrate the feature into `pbl.c`.
4. Commit the updated system to the repository.
